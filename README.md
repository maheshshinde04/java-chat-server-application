# java-chat-server-application
Java Client-Server Chat Application using Sockets and Multithreading for real-time communication between multiple users.

#Features-

- Real-time messaging

- Multiple client support
  
- Java Socket Programming
  
- Multithreading implementation
  
- User connection management
  
- Console-based communication

#Technologies Used-

- Java
  
- Socket Programming
  
- Multithreading
  
- TCP/IP Networking

#Outcomes-

- Understanding Client-Server Architecture
  
- Concurrent Programming with Threads
  
- Network Communication Concepts
  
- Real-Time Application Development
